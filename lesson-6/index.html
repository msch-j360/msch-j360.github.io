6. Getting a bit more complicated.

Goal To start to build more complicated sites. I'll give the you most of the HTML. You have to do the rest.

due Your assignment is due Friday before midnight.

Lost? Drop any and all questions in Slack. I'll have office hours this week in Slack from 8:30 to 9:30 p.m. on Wednesday and by appointment on Thursday. Don't suffer in silence.

Another copying exercise.

Here's a screenshot of a site I coded up. You're going to try to replicate this exactly.

Click for full size.

I've coded up most of the HTML for you. But, you'll have to download and include a reset stylesheet, Skeleton (which you can download here) and your own style.css file. You will also need to go into the HTML and add "columns" to match the structure I have in the screen shot. Remember, you have 12 columns to work with.

I'm also using three Google Fonts: Roboto Condensed (for the h2s), Lora (for the h3/h4s) and Lato (for kickers, and other sans-serf text). The Google Fonts site has some good documentation on how to use their fonts in your projects. You'll need to figure that out.

So here's what you'll need to do.

1. Create a new github repo inside your j360 folder. Call it news-site

2. Add all the files you normal need for a project. Your index.html, css folder, and all of your stylesheets (reset, skeleton, style.css).

3. Copy the html from here. Hint: You'll have to view the page source to get the HTML. View > Developer > View Page Source

3. Go through the html and set how many columns each chunk should be using the screenshot as your guide. I'll give you the first one: On the first div with a class of "adbox" you'll want to add two classes: "twelve" and "columns". On the div with a class of main-sidebar you'll want to add two classes: "three" and "columns". And so on.

4. Once everything is in the right spot, jump into the css. Start at the top, and style things all the way down.

5. When yours matches mine, commit your changes to github. Create a gh-pages branch and push your changes up to to make them live.

That's all guys. Good luck and let me know if you're stuck. Due on Friday at midnight.
