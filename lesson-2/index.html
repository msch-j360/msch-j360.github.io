The Assignment

For this assignment, you can keep everything in one column, stacked on top of each other. But if you’re feeling ambitious, you can try to float things next to each other. It’s a little tricky, but this piece on floats and clears lays it out pretty well.

This week’s assignment is deceptively simple. We’re going to strip all of the style away from your portfolio site and build it back up — with a few catches. You can only use black, white or any shade of gray, no other colors. You can only use the font “Georgia”, at 16 pixels, no other fonts or sizes. No images. You can use lines, borders, shades of gray, spacing, text styles (bold, italic, letter-spacing, line-spacing, alignment), width, height, etc. But no color, and only Georgia at size 16.

Your portfolio site needs to include

  • your name
  • a few sentences about you
  • a list of projects, each project should include:
    • date
    • title
    • short description
(You can use these three project descriptions, too. They’re all projects were going to do. They’ll eventually link to those projects.)

Things to think about.

What can I do given these limitations? What design options do I have? What’s most important? How can I signal to users what that is, given these limitations?

What now?

1. Sketch out your plan. Diagram what your index file should look like, including what the divs should be named and how everything should be grouped.

2. Code up your new content in your index.html file.

3. Add a reset stylesheet. This one from Eric Meyer is good. You’ll want to download it (or copy and paste it) into a file called reset.css, and link it in your <head>.

4. Delete the junk out of your style.css.

5. Add this to your css to style everything 16px Georgia. 

body {
  font-family: georgia, serif;
  font-size: 16px;
}

6. Start adding some styleeeeeee.

7. Once your happy with it, push it up to github.
